# Calendrier en rust
Pour tester, cloner le repo et taper 
```bash
cargo run -- <numéro d'un jour> <numéro d'un mois>
```

ou plus simplement (mois et année aujourd'hui)
```bash
cargo run
```

ou bien 
```bash
cargo build
./target/debug/cal <numéro d'un jour> <numéro d'un mois>
```

ou plus simplement
```bash
cargo build
./target/debug/cal
```
